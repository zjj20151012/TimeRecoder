# Rainbow
一个用Swift语言编写的语音日记小项目，Rainbow用语音记录您生活的每一天

## 兼容性
本项目采用Xcode 9.0 以及 Swift语言编写

## 主要功能
1. 用语音记录日记
2. 列表展示已经记录的日记
3. 查看播放记录的日志
4. 根据星期改变播放时cell颜色(彩虹色7种)
5. 本地存储语音信息(归档)
6. 绘制语音录制动画、播放音频动画
7. 以天为单位归类语音日记
8.
