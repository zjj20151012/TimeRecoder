# Rainbow
一个用Swift语言编写的语音日记小项目，Rainbow用语音记录您生活的每一天

## 兼容性
本项目采用Xcode 9.0 以及 Swift语言编写

## 主要功能
1. 用语音记录日志
2. 列表展示已经记录的日志
3. 查看记录的日志
4. 根据星期改变颜色
5. 本地存储日志信息
6. 绘制🌈
7. 录制音频动画
